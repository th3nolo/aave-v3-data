Aave V3 Data API

Available Endpoints:

GET /aave_v3_data.json
    → Full Aave V3 protocol data across all networks

GET /api/v1/all
    → Alias for full dataset

GET /api/v1/networks
    → List of supported networks

Usage Examples:

# Get all data
curl https://th3nolo.github.io/aave-v3-data/aave_v3_data.json

# Get specific network (using jq)
curl -s https://th3nolo.github.io/aave-v3-data/aave_v3_data.json | jq '.ethereum'

# Get specific asset
curl -s https://th3nolo.github.io/aave-v3-data/aave_v3_data.json | jq '.polygon.reserves[] | select(.symbol=="USDC")'
